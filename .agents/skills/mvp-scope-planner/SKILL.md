---
name: mvp-scope-planner
description: Plan and prioritize tasks for MVP delivery with clear acceptance criteria.
---

**Trigger:** “What should we build next for demo/MVP?”  
**Inputs:** product goal, deadline, constraints  
**Steps:**
1. Split into P0/P1/P2
2. Define acceptance criteria per chunk
3. Sequence dependencies
4. Estimate risk/time
**Output:** prioritized task board  
**Done when:** each task has owner, scope, and pass/fail criteria.
